для тега img:
1 тегу img прописываем width и height, если верстаем по макету, то указываем с десктопной версии. У картинки в css прописываем max-width: 100% и height: auto.
2 именовать картинки увеличенных в n раз название_nx. в теге img после src указываем следующий атрибут srcset, где по возрастанию перечисляем картинки. начинать с _2x.
3 используем picture внутри которого будет source n количество и img для десктопа. в source есть атрибут srcset для него нужно лишь 1 значение. тег picture используем если есть картинки для других устройств.
4 используем source в picture атрибут type для указания картинок с более легковесным форматом avif, webp, jpg.
5 если нам нужно нажать на кнопку, чтобы потом появилась картинка, лучше эту картинку пред загрузить через link. Также в link используем атрибут imagesrcset.
6 если есть картинки для пред загрузки под разные устройства, то используем media и несколько link.
7 если по макету рядом с картинкой есть подпись используем figure и figcaption

для тега :

без конкретного тега:
1 если в тег button вложена картинка и не подразумевается наличие текста, то текст все равно пишем и пишем действие кнопки. текст скрываем с помощью css. Это касается не только кнопок, но и любых элементов, где есть только картинка, без пояснения.
2 текст отличный от <html lang="название языка"> должен быть обёрнут в span с атрибутом lang="название языка"
3 при появлении на сайте модального окна, перемещение с помощью tab должно быть строго по модальному окну. Также модальное окно должно иметь кнопку его закрытия.
4 если у section есть заголовок h2, то h2 следует указать id и в section указать атрибут aria-labelledby="название-id".
5 для уведомление вместо div лучше использовать output.
